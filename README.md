# Vue Template

This is a template for Vue.js projects. It includes the following:

- Vue.js
- Vue Router
- Pinia
- Tailwind CSS
- ESLint
- Prettier
- VueUse
- Vue Devtools
- Oh, Vue Icons!

## Project setup
```bash
npm install
# or
yarn
```

### Run the project in development
```bash
npm run dev
# or
yarn dev
```
